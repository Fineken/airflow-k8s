## Создаем образ Docker

```
cd scripts/
docker сборка докера -t airflow .
```

## Развернуть в kubernetes

1. Найти $IMAGE в репозитории
2. Найти $TAG в репозитории
2. Изменить его на URL-адрес изображения и тег docker соответственно

```
cd/
./kube/deploy.sh -d с
```
